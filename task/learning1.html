website

A website is a collection of web pages that are limged together and can be accessed through the internet.

WWW

WWW stands for World wide web. it is a system of interlinked hypertext doucuments that are accessed through the internet

HTTPs

HTTPs stands for Hypertext Transfer Protocol Secure. it is a protocol used to secure communication over the internet

Client-server

Client-server is a modan for communication berween two computer programs. In this model,the client sends a request to the server, and the server send a response back to the client.

Domain

A domain is a unique name that identifils a website on the internet

Host

A host is a computer that is connected to the internet and provides to other computer on the network.

Hosting

Hosting is the act of providing a physical location for a website on a server so that it can be accessed over the internet.

FTP

FTP stands for file Transfer protocol which is a standaed network protocol used to tranfer files between a client and a server on a computer network

path

A path is a string of characters used to uniquely identify a location in a directory structure. It is composed by following the directory tree hierarchy in which components, separated by a delimiting character, represent each directory

Extension

What Does Extension Mean? An extension, in computer operating systems, is a piece of software that enhances or “extends” the capabilities of a programming language or other applications. An extension adds extra features to an already working standalone application.
